## Description

Exercises for Security Example.

## Open SSL

```shell
openssl req -x509 -newkey rsa:4096 -nodes -keyout key.pem -out cert.pem -days 365
```

## Process

Install npm packages ⬇

```shell
npm install
```

Run node application 🏎️💨

```shell
npm start
```

```shell
npm run dev
```

Clean packages 🧹🪣

```shell
npm run clean
```

Check vulnerabilities 🤔

```shell
npm audit
```

Fix vulnerabilities if needed ✅

```shell
npm audit fix
```
